# React Plugin

React and Next.js best practices, performance optimization, and composition patterns. Skills sourced from [Vercel Agent Skills](https://github.com/vercel-labs/agent-skills).

## Installation

```bash
claude plugin install react@elias-plugins
```

## Skills

### react-best-practices

40+ React and Next.js performance optimization rules across 8 categories, from eliminating waterfalls to advanced patterns. Each rule includes incorrect vs. correct code examples.

### composition-patterns

React composition patterns for building scalable components. Covers compound components, state lifting, context providers, and explicit variants over boolean props.
